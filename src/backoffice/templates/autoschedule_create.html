Create new AutoSchedule for EventType {{ event_type }}?

Creating an AutoSchedule object is the first step in using the autoscheduler. Creating an AutoSchedule will also create all the related AutoEvent and AutoSlot objects used by the autoscheduler, which might take a while (up to 30 secs in dry weather). The related objects can later be re-created and the schedule can be re-calculated as many times as needed.

Note: Creating a schedule will not affect the published EventInstances in any way. The schedule has to be applied before any EventInstances are touched. It is completely safe to play around with creating and calculating new schedules. Only applying a schedule changes EventInstances!
